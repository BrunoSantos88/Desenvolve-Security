# Commandos avançados de logs

* Verificar o logs
````
git log
git log --oneline
git log -p
git log --pretty 
git log --format
git log --format="%H %an"
````


# Vendo as alterações

````
git show
git show id
git show --help
````

# Status verificar antes de alterar

````
git status
git diff
git diff --help
````
